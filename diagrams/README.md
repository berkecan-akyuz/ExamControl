# System Diagrams

This directory contains visual documentation for the system architecture.

## Files
*   **`erd.mermaid`**: Entity Relationship Diagram. Shows the SQL schema structure.
*   **`use_cases.puml`**: UML Use Case diagram for Admin and Proctor actors.
*   **`sequence_checkin.mermaid`**: Sequence diagram for the Check-in flow.
*   **`activity_diagram.mermaid`**: Flowchart of user activities.

## Tools
*   **Mermaid**: Rendered by GitHub natively.
*   **PlantUML (.puml)**: Requires PlantUML viewer.
